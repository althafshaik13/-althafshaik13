## Featured Projects

### [solvit-platform](https://github.com/althafshaik13/solvit-platform)
Real-time life problem solver — post any emergency (burst pipe, car breakdown, legal issue) and get **AI-generated actionable steps instantly**, matched with a **verified local solver**, pay via **Razorpay escrow** released only after resolution. Built as 6 Spring Boot microservices in one Maven multi-module repo.
`Spring Boot 3.2` `Java 17` `Keycloak` `OAuth2` `Razorpay` `Redis` `PostgreSQL` `Docker` `Kubernetes`

---

### [payment-gateway-service](https://github.com/althafshaik13/payment-gateway-service)
Production-grade payment microservice with **Razorpay** integration. Pluggable gateway abstraction, HMAC-SHA256 webhook verification, payment event audit trail, and a mock mode for dev/test without API keys.
`Spring Boot` `JPA` `PostgreSQL` `Docker`

---

### [ecommerce-microservices](https://github.com/althafshaik13/ecommerce-microservices)
3-service e-commerce platform (User, Product, Order) behind a **Spring Cloud Gateway**. Includes Docker Compose for local dev and Kubernetes manifests for deployment.
`Spring Boot` `Spring Cloud Gateway` `PostgreSQL` `Docker` `Kubernetes`

---

### [auth-service](https://github.com/althafshaik13/auth-service)
**OAuth2/OIDC** resource server backed by **Keycloak**. JWT validation via JWKS endpoint, role-based access control (ADMIN / DOCTOR / PATIENT), fully containerised.
`Spring Security` `Keycloak` `OAuth2` `JWT` `Docker`

---

### [notification-engine](https://github.com/althafshaik13/notification-engine)
Multi-channel dispatcher for **SMS** (MSG91), **Email** (SMTP), and **Push** (FCM). Async dispatch with `@Async`, automatic retry with `@Retryable`, dead-letter queue, and a delivery audit log per attempt.
`Spring Boot` `Spring Retry` `Strategy Pattern` `PostgreSQL` `Docker`

---

### [appointment-booking](https://github.com/althafshaik13/appointment-booking)
Doctor appointment scheduler with **Redis**-backed slot caching and **optimistic locking** (`@Version`) to prevent double-booking under concurrent load. Full payment confirmation flow included.
`Spring Boot` `Redis` `PostgreSQL` `Docker`
